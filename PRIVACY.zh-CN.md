# 隐私政策

[**English**](https://github.com/gohj99/Wear_Translator_Public/blob/master/PRIVACY.md)

**最后更新日期：2025年12月14日**

本隐私政策旨在告知您（用户），当您使用我们的 **腕间翻译** 软件（以下简称“软件”）时，**gohj99**（以下简称“我们”）如何处理您的信息。

请在使用本软件前仔细阅读本隐私政策。使用本软件即表示您理解并同意本政策中所述的条款。如果您不同意本隐私政策中的条款，请勿使用本软件。

---

## 我们收集的信息

**腕间翻译 不会收集任何可用于识别您个人身份的信息，例如您的姓名、电子邮箱地址、联系人或账户凭证。**

但是，为了确保服务能够可靠、安全地运行，本软件会处理少量**设备层面的技术标识符**，具体包括：

### Android 设备标识符（Android ID）

本软件会访问 Android 系统提供的 **Android ID**。
该标识符仅用于以下目的：

* 防止翻译服务被滥用（例如自动化请求、机器人或异常频繁使用）
* 实施服务使用限制并维持公平使用
* 检测欺诈或恶意行为
* 维持服务稳定性与服务质量
* 在同一设备上提供一致的服务功能体验

### 重要说明

Android ID：

* **不会用于广告用途**
* **不会用于用户画像或个性化分析**
* **不会用于跨应用或跨网站跟踪您的活动**
* **不会与广告商或数据经纪商共享**
* **无法用于识别您的真实身份**

我们仅将该标识符作为区分访问本服务设备的技术手段使用。它被视为一种**安全与服务完整性措施**，而非个人身份识别信息。

## 我们不会收集的信息

我们**不会收集或存储**以下信息：

* 您的姓名
* 电子邮箱地址
* 电话号码
* 联系人信息
* 位置信息
* 对话内容
* 语音录音
* 翻译历史记录
* 个人文件
* 账户凭证

您的翻译输入内容不会被我们保留。

## 翻译服务提供商

本软件的翻译功能由第三方服务提供商 **Microsoft Azure 翻译服务** 提供支持。当您在 **腕间翻译** 中输入文本或语音进行翻译时，这些数据将直接发送到 Microsoft Azure 的翻译接口进行处理。

请注意：

* **我们不会存储或访问您的翻译内容。**
* Microsoft Azure 有其自身的隐私政策，我们建议您仔细阅读。您可以通过访问 Microsoft 官方网站查找与 Azure 服务相关的隐私信息。

## 剪贴板使用

**腕间翻译** 可能会读取或写入您设备的剪贴板，以便复制或粘贴翻译内容。所有剪贴板操作均**完全在您的设备上**执行，我们或任何第三方**绝不会传输、上传或存储**您的剪贴板数据。这确保您的剪贴板数据始终保持私密和安全。

## 数据安全

由于我们不收集或存储任何用户数据，因此我们的系统不存在数据泄露风险。数据传输（例如，传输至 Azure 翻译服务）遵循行业标准安全协议。
